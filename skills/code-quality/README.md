# Code Quality Skill

A comprehensive skill to ensure Flutter applications remain maintainable and performant as they grow.

## Description
This skill enforces professional-grade Dart and Flutter patterns. It focuses on reducing widget complexity, optimizing the build tree, and ensuring that the project's `analysis_options.yaml` is the primary driver for code style.

## Key Principles
- **Readability:** Named parameters and small widget classes.
- **Scalability:** Grouping parameters into classes and utilizing controllers.
- **Performance:** Avoiding helper functions and isolating rebuilds.

## Usage
Add this skill to your project and prompt:
> "Refactor the following widget for better code-quality."
> "Generate a new screen following the code-quality skill requirements."
